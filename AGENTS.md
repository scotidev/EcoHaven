# Diretrizes Unity + C# Dev Kit

- **Ambiente**: Estou usando o C# Dev Kit e a extensão oficial da Unity.
- **Versão do SDK**: O projeto depende do .NET 8 SDK ou superior.
- **Sincronização**: Se você criar ou renomear arquivos .cs, lembre-me de "Regenerate project files" no Unity para atualizar o Visual Studio Editor v2.0.20+.
- **Debug**: Para depurar, use o comando "Attach Unity Debugger".
- **Documentação**: Se eu tiver dúvida sobre uma API da Unity, você pode me sugerir usar o comando "Unity API Reference".
- **Eficiência**: Nunca tente ler pastas de sistema da Unity (Library, Temp, Packages). Foque apenas na pasta Assets para lógica de jogo. Lembre de ver o .clineignore
